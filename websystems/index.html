If you don't already know, My name is Ryan Soury

I am currently an active undergraduate proceeding through my Bachelor of Science in Information and Technology. Other student usually just refer to it as the "Biscuit" course. I learnt some Bash Script in this course, and actually use it whenever I program side projects. I used to depend heavily on Gulp or NPM scripts to run even the smallest tasks, but Bash is cool to bundle a bunch of CLI commands into a single task. One example of this is to setup my git remote repos so that when I push code, I call one command and it stores both on Gitlab and Github.

I am currently employed at ZipMoney Limited. It is a financial technology business that provides loans on checkout, so that you can "Buy Now, Pay Later". I specifically work on integrating the API into merchant websites and applications. I like how free the environment is over there, they let me study and work simulatneously and if necessary allow for me to work remotely, however in some circumstances it does become very challenging and requires my full focus.
