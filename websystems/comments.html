Structure

The website is structured primarily with tables and table cells, similarly to the way Marketing Emails are structured today because of the requirement to use HTML4 and CSS2 which do not have any sort of responsive grid based functionality like CSS3's Flexbox or @media selectors. Within each of these cells is our content, usually wrapped in a parent container. For any sort of list, for example a list of anchors, we used the unordered list and its list items elements to ensure that the structure and understanding of the list is clear in the HTML markup.

The only id I used was for the main image which is the only image of its kind. Ids are good to specifically identify an element throughout the whole website. In CSS3, Ids are also good to behave as starting points in long CSS Selectors. 

                      
                    
                      #introPicture{
                        border: 6px dashed #8c6eff;
                      }

I used many classes for even the smallest CSS styles, however there are primary classes that were used many times in one webpage which is essentially what classes are for. Classes are reusable labels representing a set of styles. When applied to an element, the element inherit the styles defined in the CSS class.

Class names are meant to be self explanatory. In the example below, it is clear that .main-nav-anchor is a class abbreviation for "Main Navigation Anchor" which is correct described as it defines the styling for the main navigation anchor elements on the web page. As already described, the reason why they're classes ".main-nav-anchor" and not ids "#main-nav-anchor" is because classes are more for style reusability and because multiple elements with the same id contradicts the purpose of an identification attribute. 

                      .main-nav-anchor{
                        text-decoration: none;
                        color: #8c6eff;
                        border: 1px solid #8c6eff;
                        display: inline-block;
                        font-size: 18px;
                        width: 200px;
                        height: 40px;
                        line-height: 2;
                      }
                      .border-bottom-cell{
                        border-bottom: 5px solid rgb(240, 240, 240);
                      }
                      .border-right-cell{
                        border-right: 1px solid rgb(220, 220, 220);
                      }
                      .home-nav{
                        width: 40px;
                        vertical-align: middle;
                      }
                      .home-img{
                        margin: 5px 0;
                      }
